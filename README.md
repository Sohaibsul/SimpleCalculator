# 🧮 Advanced Python Web Calculator

## 📖 Overview
This is a simple, fast, and interactive web-based calculator built using Python and Streamlit. It was designed to handle both basic arithmetic and slightly more advanced mathematical operations in a clean, user-friendly interface. 

## 🚀 Live Demo
You can try out the calculator right now directly in your browser!
(https://huggingface.co/spaces/SOHAIBSUL123/SimpleCalculator)



## ✨ Features
This calculator supports the following operations:
* **Basic Math:** Addition (`+`), Subtraction (`-`), Multiplication (`*`), and Division (`/`) with built-in zero-division protection.
* **Advanced Math:** Square (`x²`), Exponents (`xʸ`), and Base-10 Logarithms (`log10`).
* **Smart UI:** The interface dynamically updates based on your selected operation. For example, if you choose "Square" or "Logarithm", it automatically hides the second number input since those operations only require one number.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** Streamlit (for the web interface)
* **Libraries:** Python's built-in `math` module (for logarithmic calculations)
* **Hosting:** Hugging Face Spaces
